# JavaScript Learning Projects

This repository contains my learning projects as I climb the ladder of JavaScript fundamentals and web development.

## Projects Included

### 1. Dynamic Background Selector
A simple project to practice DOM manipulation and event handling in JavaScript.
- **Features**: Select different background colors dynamically
- **Skills Practiced**: DOM manipulation, event listeners, CSS integration

### 2. Kanban Project
A task management board with drag-and-drop functionality.
- **Features**: 
  - Add tasks to different boards (To-Do, In Progress, Done)
  - Edit existing tasks
  - Delete tasks
  - Drag tasks between boards
  - Create new custom boards dynamically
  - Task counter for each board
- **Skills Practiced**: DOM manipulation, event handling, drag-and-drop API, local state management, dynamic element creation
